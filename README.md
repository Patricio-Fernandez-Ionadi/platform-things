# Phaser practices

## Player controls

- Horizontal movement: keyboard arrows ← →
- Jump: keyboard arrow ↑
- Duck: ↓
- Draw/Shealth weapon: shift
- Slice:
  - direction + arrow down
  - ← | → + ↓
- Jump roll:
  - direction + jump + spacebar
  - ← | → + ↑ + space
- Attacks:
  - Swipe blade Up: Q
  - Swipe blade Down: W
  - Swift: E
